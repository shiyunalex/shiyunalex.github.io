CAM and Grad-CAM

Grad-CAM:Visaul Explainations from Deep Networks via Gradient-based Localization
Learning Deep Features for Discriminative Localization

CAM (Class Activation Map)

Model Structure

CNN -> GAP (Global Average Pooling) -> Dense (n-class output)

More detailed,

328 channels -(GAP)-> tensor(328,) -(dense)-> tensor(8,)

GAP layer could be replaced by Global Max Pooling.

How to calculate CAM

  1. 2D-GAP (torch.nn.2D_Average_Pooling)

For a given model, let $f_k$ represent the value of the $k_{th}$ channel at spatial location $(x,y)$. $F_k$ is the $k_{th}$ output of GAP layer.(let $Z$ represent the number of unit in $k_th$ chanel, $F_x$ should be divided by $Z$,$Z$ is omitted as it is a contant )

  1. CAM

$M_c$ , the spatial element of class $c$ activation map at (x,y)

Class score $S_c$ , the input of softmax layer for class $c$.

Grad-CAM

The restrict of CAM: must use GAP as the last second layer of the model, which often makes the model sacrifice in accuracy.

The idea of Grad-CAM is to find the weight $w$ ,which has the same functionality as the last dense layer in CAM.

3D CNN machining feature recognition

3D convolutional neural network for machining feature recognition with gradient-based visual explanations from 3D CAD models



Status quo:

Conventional methods

  1. Graph based
  2. Volume decomposition
  3. Hint based
  4. Similarity based
    • generate the axis of ratation using moment of inertia
    • using axis of rotaion to generate the distribution

Cons:

  • performce deteriorates when noise occurs.

Deep learning based methods

  1. Rotionally invariant model
  2. Learning both 3D data and 2D muti-view data
  3. FeatureNet(3D-CNN)

Class activation mapping for feature location

  1. Using GAP (global average pooling)
  2. Grad-CAM (using the gradient of backpropagation)



Related model: FeatureNet

  • Has good performance on single feature recognition task, performance on multi-feature recognition task is not distinguished.
  • Is not able to locate the feature.



🤗Paper model:

  • Part1: 3D CNN, for multi-machining feature classification.
  • Part2: Grad-CAM, for feature area estimation.
  • Achived 98% accuracy on feature classification.
